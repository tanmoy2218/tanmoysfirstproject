Table Basics: <table>, <tr>, <th>, <td>

The <table> Element

The container for all table elements. Defines that content should be rendered as a table.

<table>
  <!-- table content goes here -->
</table>

Table Rows: <tr>

Defines a row within the table. Each <tr> contains either <th> or <td> elements.

<tr>
  <td>Cell 1</td>
  <td>Cell 2</td>
</tr>

Table Headers: <th>

Defines a header cell. Used for column or row headers. Screen readers use these to provide context.

<tr>
  <th>Product Name</th>
  <th>Price</th>
  <th>Stock</th>
</tr>

Table Data: <td>

Defines a standard data cell within the table.

<tr>
  <td>Laptop</td>
  <td>$899.99</td>
  <td>15</td>
</tr>

Basic Table Example

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>Country</th>
  </tr>
  <tr>
    <td>John</td>
    <td>28</td>
    <td>USA</td>
  </tr>
  <tr>
    <td>Maria</td>
    <td>32</td>
    <td>Spain</td>
  </tr>
</table>

Semantic Table Structure: <thead>, <tbody>, <tfoot>

Table Head: <thead>

Contains header content (column headings). Defines the header section of a table.

<thead>
  <tr>
    <th>Product</th>
    <th>Price</th>
    <th>Quantity</th>
  </tr>
</thead>

Table Body: <tbody>

Contains the primary data of the table. There can be multiple <tbody> elements to group rows.

<tbody>
  <tr>
    <td>Keyboard</td>
    <td>$49.99</td>
    <td>23</td>
  </tr>
  <tr>
    <td>Mouse</td>
    <td>$25.99</td>
    <td>37</td>
  </tr>
</tbody>

Table Foot: <tfoot>

Contains footer content such as sums, totals, or summary information.

<tfoot>
  <tr>
    <td>Total</td>
    <td>$75.98</td>
    <td>60</td>
  </tr>
</tfoot>

Complete Semantic Table Example

<table>
  <caption>Monthly Sales Report</caption>
  <thead>
    <tr>
      <th>Product</th>
      <th>January</th>
      <th>February</th>
      <th>March</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Widget A</td>
      <td>150</td>
      <td>210</td>
      <td>185</td>
    </tr>
    <tr>
      <td>Widget B</td>
      <td>90</td>
      <td>95</td>
      <td>110</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Total</td>
      <td>240</td>
      <td>305</td>
      <td>295</td>
    </tr>
  </tfoot>
</table>

Merging Cells: colspan and rowspan

Column Span: colspan

Merges cells horizontally. The value indicates how many columns the cell should span.

<tr>
  <th colspan="2">Personal Information</th>
</tr>
<tr>
  <td>First Name</td>
  <td>Last Name</td>
</tr>

Row Span: rowspan

Merges cells vertically. The value indicates how many rows the cell should span.

<tr>
  <td rowspan="2">Category A</td>
  <td>Item 1</td>
</tr>
<tr>
  <td>Item 2</td>
</tr>

Complex Table with Merged Cells

<table>
  <tr>
    <th rowspan="2">Department</th>
    <th colspan="2">Employees</th>
  </tr>
  <tr>
    <th>Full-time</th>
    <th>Part-time</th>
  </tr>
  <tr>
    <td>Marketing</td>
    <td>12</td>
    <td>5</td>
  </tr>
  <tr>
    <td>Development</td>
    <td>25</td>
    <td>8</td>
  </tr>
</table>
