Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage.

HyperText

This refers to the ability to create links that connect web pages to one another, making the web a connected "web" of information.

Markup Language

This means you use "tags" to surround your content, giving that content meaning and structure. You are "marking up" a plain text document.

1. Headings: <h1> to <h6>

Headings are tags used to define titles and subtitles on your page. They are crucial for creating a logical hierarchy and outline for your content.

Heading Hierarchy:

  • <h1>: The most important heading, typically used only once per page for the main title.
  • <h2>: A major section heading.
  • <h3>: A sub-section heading under an <h2>.
  • ...and so on, down to <h6>, the least important heading.

Purpose:

To structure your document in a way that is understandable to both humans and machines (like search engines and screen readers). It is not just for making text big; it is for giving the text structural importance.

Example:

<h1>Main Title of the Page</h1>
<h2>Section Heading</h2>
<h3>Sub-section Heading</h3>

2. Paragraphs: <p>

The paragraph tag is the most common tag you'll use. It's for grouping sentences and blocks of text together.

Purpose:

To define a distinct paragraph of text. Browsers automatically add a little bit of space before and after a <p> element, separating it from other content. You should not use multiple line breaks; you should use multiple <p> tags.

Example:

<p>This is the first paragraph. It contains several sentences about a topic.</p>
<p>This is a second, separate paragraph. The browser will display it with a space between it and the first one.</p>

3. Horizontal Rules: <hr>

The <hr> tag defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.

Purpose:

The <hr> element is used to separate content (or define a change) in an HTML page.

Example:

<p>This is the first topic.</p>
<hr>
<p>This is a completely different topic.</p>

4. Line Breaks: <br>

This tag tells the browser, "Stop writing on this line and immediately start on the next one." It's like hitting the Enter key once in a poem or an address.

Purpose:

Creates a single line break.

Example:

<p>221B Baker Street<br>
London, England</p>

5. Lists: <ul>, <ol>, <li>

These tags are used to create lists. This is a perfect example of nesting.

List Types:

  • <ul>: An Unordered List. It creates a bulleted list.
  • <ol>: An Ordered List. It creates a numbered list.
  • <li>: A List Item. Each item in either type of list must be wrapped in an <li> tag.

Purpose:

To group related items together in a list format.

Example (Nesting in action):

<ul>
  <li>Tea Bag</li>
  <li>Water</li>
  <li>Sugar</li>
  <li>Milk</li>
</ul>

<ol>
  <li>First, boil the water in a kettle.</li>
  <li>Add tea bag, sugar and milk into it</li>
  <li>Keep boiling it for 5 minutes </li>
  <li>Serve the tea by pouring it into cup</li>
</ol>

6. Links: <a>

The Anchor tag is what makes the web "hypertext." It is used to create a hyperlink to another webpage, a file, or a location within the same page.

Purpose:

To make text (or an image) clickable, allowing users to navigate. It requires an attribute called href (hypertext reference) to specify the destination URL.

Example:

<a href="https://www.coderarmy.in/">Visit Coder Army</a>

The target Attribute:

By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.

Target Values:

  • _self - Default. Opens the document in the same window/tab as it was clicked
  • _blank - Opens the document in a new window or tab

Example with target:

<a href="https://www.coderarmy.in/" target="_blank">Visit Coder Army (opens in new tab)</a>

7. Images: <img>

The Image tag is a self-closing tag used to embed an image onto your page.

Purpose:

To display a visual image. It requires two main attributes:

  • src (source): The path or URL to the image file. This is mandatory.
  • alt (alternative text): A description of the image. This is vital for accessibility (screen readers for the visually impaired will read this out) and for when the image fails to load.

Example:

<img src="https://cdn.britannica.com/16/234216-050-C66F8665/beagle-hound-dog.jpg" 
     height="300px" 
     width="300px" 
     alt="A beautiful Beagle dog">

Optional Attributes:

  • height: Sets the height of the image (in pixels or percentage)
  • width: Sets the width of the image (in pixels or percentage)

A simple example by using all these attributes:

<!DOCTYPE html>
<html lang="en">
<head>
    <title>HTML Basics</title>
</head>
<body>
    <h1>My Website</h1>
    <p>Welcome to my page!</p>
    <hr>
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
    </ul>
    <a href="https://www.coderarmy.in" target="_blank">Visit CoderArmy</a>
    <br>
    <img src="https://www.pngfind.com/pngs/m/597-5976720_html5-logo-lenguaje-de-programacion-html5-hd-png.png" alt="Logo" width="100">
</body>
</html>
